# Mekanism-The-Valinest
This is the repository for modcraft.

# Setup
1. Clone the repository to your local machine.
2. Open the project in your preferred IDE (e.g., IntelliJ IDEA, Eclipse).
3. Run the task "deobfJar" in the animation injector and make the dir "libs" in the root of this project and move the generated jar from "build/libs" to "libs".
4. Run the task "genIntelliJRuns"
5. Run the "runClient" configuration to start the development environment with the mods.
